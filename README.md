# Sys-chan
Бот для управления сервером через ssh из чата дискорда.


# Файл конфигурации
config.json — создать в корне.

```json
{
    "bot_token" : "токен бота дискорда",
    "ssh" : {
        "host" : "ip хоста",
        "port" : 22,
        "user" : "юзверь",
        "key" : "путь к файлу ключа"
    }
}
```
